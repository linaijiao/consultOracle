文件结构说明

1. 十神解读（核心部分）

每个十神包含：

personality: 核心性格特征

detailedAnalysis: 详细性格分析

strengths: 优点列表

weaknesses: 缺点列表

careerSuggestions: 职业建议

compatibility: 相生相克关系

luckAnalysis: 运势分析


2. 五行解读

每个五行的性格特征

生克关系说明


3. 旺衰程度解读

五种旺衰状态的含义说明




特点说明

1. 全面性

覆盖十神系统的所有方面

包含性格、职业、运势等多维分析

提供相生相克关系指导


2. 实用性

语言通俗易懂

建议具体可行

便于CC在UI中直接显示


3. 专业性

基于传统命理学理论

结合现代心理学理解

保持文化传统的准确性


4. 扩展性

结构清晰，便于后续添加更多内容

支持多语言扩展（如有需要）
